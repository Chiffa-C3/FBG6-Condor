# Проект печатной головы "Кондор" 

<img src="./PIC/pic1.png" alt="pic1" width="100%" height="100%" />

Проект печатной головы для принтера FBG6 с поддержкой [Klicky-Probe](https://github.com/jlas1/Klicky-Probe) и фидером на основе Mini Stealthburner с быстрым отжимом филамента для более легкой замены прутка.

Используемые детали:
- Стоковый радиатор FBG6
- Гладкое биметалическое горло 21мм
- Турбинки 4010 GDSTime
- Вентилятор 3010 GDSTime
- Набор BMG шестерней [пример](https://aliexpress.ru/item/1005003334389217.html)
- Шаговый двигатель Nema 14 36STH20 [пример](https://aliexpress.ru/item/1005002717890067.html?sku_id=12000031330274970)
- 8шт Круглые магниты 5х5мм
- Микрик [пример](https://aliexpress.ru/item/4001033375208.html)
- Вплавляемые резьбовые латунные вставки М3 4-5мм длины [пример](https://aliexpress.ru/item/1005005372220229.html)
